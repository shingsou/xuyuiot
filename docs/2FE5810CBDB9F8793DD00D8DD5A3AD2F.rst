物联卡
=================
.. Note::

	集三大运营商专为物联网应用提供的流量卡，相比传统的流量卡，物联卡拥有资费低、安全、通信效率高等特点。 解决方案涵盖：车联网、共享单车、视频监控、城市智能路灯、pos机、警用巡逻车、自动售卖机等。
